Awards Won


BAFTA Games Awards - Best game of 2020 (Apex Hunters)

The British Academy Video Games Award for Best Game is an award presented annually by the British Academy of Film and Television Arts (BAFTA). It is given in honor of "the best game of the year,across all genres and platforms".Our game Apex Hunters won it this year.






Game Awards 2019 - Best shooting game 2019 (Call of work)

The Game Awards 2019 was an award show that honored the best video games of 2019.The Game Awards 2019 were held at the Microsoft Theater in Los Angeles.Our game Call of work won the best shooting game of 2019 award.






The Game Developers Choice Awards - Best Android game 2019 (Virtual Cricket)

The Game Developers Choice Awards are awards annually presented at the Game Developers Conference for outstanding game developers and games. Our game Virtual Cricket won the best Android game of 2019 award.
